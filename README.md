# Data Visualization Projects (Matplotlib & Seaborn)

## Overview

This repository contains exploratory data analysis and visualization projects using Python libraries Matplotlib and Seaborn. The goal is to understand real-world datasets and extract meaningful insights through visual storytelling.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Projects Included

* Basic plotting techniques (bar charts, histograms, line plots)
* Advanced visualization using Seaborn
* Dataset-based analysis with insights

## Key Learnings

* Understanding data distributions
* Identifying patterns and trends
* Creating meaningful visual representations of data

## Datasets

All datasets used are included in the `datasets/` folder for reproducibility.
